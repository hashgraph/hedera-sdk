# class `LiveHashQuery`

Retrieve the latest state of a topic.

This method is unrestricted and allowed on any topic by any payer account.

Deleted accounts will not be returned.

Returns [`LiveHash`](./LiveHash.md) from [`execute`](../Query.md).
